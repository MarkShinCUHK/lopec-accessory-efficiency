# 로펙 찐 악세 효율 계산기

내 캐릭에 맞는 악세 효율 찾기 - 경매장 매물별 로펙점수 상승 비교 및 시각화

## 왜 만들었나

로스트아크에서 악세를 바꿀 때 중요한 건 단순히 옵션 등급이 아니라, **내 캐릭터에 장착했을 때 로펙점수가 실제로 얼마나 오르는지**입니다.

기존 도구들도 도움이 되지만 아쉬운 점이 있었습니다.

- ALOA 악세 검색은 보기 편하지만, 내 캐릭 기준 로펙점수 상승분을 바로 보여주지는 않습니다.
- LOPEC 시뮬레이터와 크롬 익스텐션은 점수 확인에 좋지만, 악세사리 매물들을 가격과 함께 한 번에 비교하기는 어렵습니다.
- 특히 “옵션 조건만 맞는 제일 싼 악세”가 아니라, **현재 올라온 매물별 점수 상승량과 골드 효율**을 보고 고르고 싶었습니다.

그래서 이 앱은 경매장 매물을 불러와 현재 캐릭터에 가상 장착해 보고, 로펙점수 상승량과 1점당 골드를 비교합니다.

## 주요 기능

- 캐릭터명으로 현재 장착 악세와 로펙점수 불러오기
- 딜러/서포터 기준 악세 효율 계산
- 목걸이, 귀걸이, 반지 경매장 매물 검색
- 옵션 타겟 검색: 원하는 상/중/하 옵션 조건으로 검색
- 가격 타겟 검색: 예산 안에서 여러 악세 교체 조합 비교
- 로펙점수 상승량, 1점당 골드, 증가율 정렬
- 표/카드 보기 전환
- 효율 그래프 시각화
- 최신 로펙 수식 일치 확인

## 실행

```bash
npm i
npm run dev
```

브라우저에서 `http://localhost:3000`을 엽니다.

## 환경 변수

`.env`에 Lostark Open API JWT를 넣습니다. `bearer`는 생략해도 됩니다.

```env
LOSTARK_API_JWT=eyJ...
```

## 검증

```bash
npm run typecheck
npm run verify:lopec
```

`verify:lopec`는 최신 로펙 번들의 수식 fingerprint와 로컬 계산 결과를 비교합니다.
